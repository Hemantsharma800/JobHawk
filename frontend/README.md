# Glixtron - Job Portal Dashboard

A modern job portal application dashboard built with React.

## Features
- Dashboard with job statistics
- Jobs management
- Portals integration
- Analytics
- User profile management

## Tech Stack
- React
- JavaScript
- CSS
- HTML

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
